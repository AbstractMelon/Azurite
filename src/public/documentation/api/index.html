API Documentation

Get Games

URL: /api/v1/games

Method: GET

Description: Retrieves a list of games.

Sample Response:


[
  {
    "id": "game1",
    "name": "Game 1",
    "description": "Description of Game 1"
  },
  ...
]

Submit Ticket

URL: /submit-ticket

Method: POST

Description: Submits a support ticket.

Sample Request Body:


{
  "subject": "Issue with mod",
  "description": "Description of the issue",
  "user": "username"
}

Sample Response:


{
  "success": true,
  "message": "Ticket submitted successfully"
}

Create Account

URL: /api/v1/createAccount

Method: POST

Description: Creates a new account.

Sample Request Body:


{
  "username": "newuser",
  "password": "securepassword",
  "bio": "This is a bio",
  "email": "email@example.com"
}

Sample Response:


{
  "message": "Account created successfully"
}

Login

URL: /api/v1/login

Method: POST

Description: Logs in a user.

Sample Request Body:


{
  "username": "existinguser",
  "password": "password"
}

Sample Response:


{
  "message": "Login successful"
}

Upload Mod

URL: /api/v1/uploadMod

Method: POST

Description: Uploads a new mod.

Sample Request Body:


{
  "modName": "New Mod",
  "modDescription": "Description of the mod",
  "modVersion": "1.0.0",
  "modFile": "path/to/mod/file",
  "modIcon": "path/to/mod/icon",
  "gameId": "game1"
}

Sample Response:


{
  "message": "Mod uploaded successfully",
  "modFileUrl": "/cdn/mods/game1/new-mod/file",
  "modIconUrl": "/cdn/mods/game1/new-mod/icon",
  "manifestUrl": "/cdn/mods/game1/new-mod/manifest.json"
}

Get Mods From Thunderstore

URL: /api/v1/getModsFromThunderstore

Method: GET

Description: Retrieves a list of mods from Thunderstore.

Sample Response:


[
  {
    "id": "modowner/modname",
    "name": "Mod Name",
    "description": "Short description of the mod"
  },
  ...
]

Get Mod Details From Thunderstore

URL: /api/v1/getModDetailsFromThunderstore

Method: GET

Description: Retrieves detailed information about mods from Thunderstore.

Sample Response:


[
  {
    "id": "modowner/modname",
    "name": "Mod Name",
    "description": "Detailed description of the mod"
  },
  ...
]

Get Mods

URL: /api/v1/mods/:gamename

Method: GET

Description: Retrieves a list of mods for a specific game.

Sample Response:


[
  {
    "name": "Mod Name",
    "description": "Description of the mod",
    "id": "mod-id",
    "version": "1.0.0",
    "modFile": "/cdn/mods/game1/mod-id/file",
    "modIcon": "/cdn/mods/game1/mod-id/icon"
  },
  ...
]
